Detailed Description

Handles the visual effects and animations for dice, including color changes, outline effects, and reveal animations.

This class is responsible for managing the visual representation of dice, such as applying colors for different states (e.g., rolling, selection), animating outline effects, and controlling the duration of dice reveal animations. It provides configurable properties for customizing these effects.
