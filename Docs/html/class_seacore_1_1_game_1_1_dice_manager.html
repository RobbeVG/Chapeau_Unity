Seacore.Game.DiceManager.Die[]

Splits the dice information into two groups based on the specified state.

Parameters
stateThe DieState flag used to determine the grouping of dice.
Returns
A tuple containing two arrays of DieInfo objects:
  • The first array contains dice that match the specified state.
  • The second array contains dice that do not match the specified state.
Both arrays are of fixed size, with unused elements set to null.
