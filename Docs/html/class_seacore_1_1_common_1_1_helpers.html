static GameObject Seacore.Common.Helpers.GetObjectFromScreen ( Vector2 screenPos,
LayerMask mask )
static

Returns the GameObject located at the specified screen position, if any, using a raycast against the provided layer mask.

Uses the current camera to cast a ray from the specified screen position. If multiple objects overlap at the position, the closest one is returned.

Parameters
screenPosThe screen coordinates, in pixels, from which to cast the ray. Typically corresponds to the mouse or pointer position.
maskA LayerMask that specifies which layers to include in the raycast. Only objects on these layers will be considered.
Returns
The GameObject hit by the raycast at the given screen position and layer mask; or null if no object is found.
